O que é o unix?
Sistema operacional que ficou muito famoso em 1969, ganhando muito variações
tornando o principal sistema operacional no mercado.

O que é o kernel?
É o core do computador, no momento em que é iniciado o computador, o kernel faz a leitura
dos hardwares para consegui tornar a interface mais amigável aos olhos humanos.

O que é o Shell?
O shell é a famosa tela preta, onde interage e manipula pastas, arquivos e comandos do unix.
